Install

Call jquery. It's recommended to call on google code. Latest version.

<script type="text/javascript" src="http://www.google.com/jsapi"></script>
<script type="text/javascript">google.load("jquery", "1.3.2");</script>

Download jQuery limitInput plugin v0.2.1. Call limitInput plugin.

<script type="text/javascript" src="jquery.limitinput.js"></script>

Add the jquery.limitInput.css at your head page.

<link rel="stylesheet" href="jquery.limitinput.css" media="all" />
